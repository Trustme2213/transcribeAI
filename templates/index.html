Инструкция по использованию
  1. Найдите бота в Telegram по имени пользователя, указанному выше
  2. Отправьте команду /start для начала работы
  3. Для использования бота необходимо получить авторизацию от администратора
  4. После авторизации используйте команду /process_audio для загрузки и обработки аудио
  5. Загрузите аудиофайл в формате MP3
  6. Дождитесь завершения обработки и ответьте на оставшиеся вопросы
  7. Получите готовый PDF-отчет
Возможности бота
  • Обработка длинных аудиофайлов
  • Автоматическое разделение на части для эффективной транскрипции
  • Транскрипция аудио с помощью Whisper
  • Анализ контента с использованием YandexGPT
  • Автоматическое заполнение анкеты на основе анализа
  • Формирование PDF-отчетов
  • Система авторизации пользователей
